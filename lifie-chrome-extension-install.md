# Lifie WebChat Widget - Chrome Extension
For demo purpose, render website chat widget on any website.

# Steps
1.⁠ ⁠Unzip the [lifie-chrome-extension.zip](lifie-chrome-extension.zip) file.

2.⁠ ⁠Go to chrome://extensions/ (if not on chrome, go to the extensions settings page of your browser).

3.⁠ ⁠Enable "Developer mode".

4.⁠ ⁠Click "Load Unpacked".

5.⁠ ⁠Select the extracted lifie-chrome-extension.zip file.

6.⁠ ⁠Pin the extension to your browser.

7.⁠ ⁠Paste the required agent's "Integration code" into the input box of the extension.

8.⁠ ⁠Enable the extension.

9.⁠ ⁠Visit any webpage to view the web chat widget.
